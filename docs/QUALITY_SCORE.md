## 3. 게임 플레이 품질 기준

### 조작감
| 항목 | 기준 | 측정 방법 |
|------|------|----------|
| 패들 입력 지연 | 16ms 이하 (1프레임) | 터치 → 패들 이동 딜레이 |
| 볼 반사각 정확도 | 물리 법칙 준수 | 입사각 = 반사각 ±2° |
| 아이템 획득 판정 | 패들 영역 100% 커버 | 엣지 케이스 테스트 |
| 볼 재시작 탭 반응 | 100ms 이내 발사 | 탭 → 볼 발사 딜레이 |

### 밸런스 기준
| 항목 | 기준 |
|------|------|
| 입문 스테이지 (1~3) 클리어율 목표 | 90% 이상 |
| 중반 스테이지 (4~7) 클리어율 목표 | 60% 이상 |
| 후반 스테이지 (8~10) 클리어율 목표 | 30% 이상 |
| 1판 평균 플레이 시간 | 90초 ~ 3분 |
| 아이템 없이 후반 클리어 가능 여부 | 불가능 (의도적) |

### 스테이지 JSON 검증 기준
에이전트가 스테이지 생성 후 자동 검증해야 하는 항목:

```
- [ ] grid_cols = 7, grid_rows = 10 준수
- [ ] 전체 벽돌 수: 20개 이상 70개 이하
- [ ] HP 범위: 구간별 기준 준수
- [ ] 아이템 드롭 비율: 구간별 기준 ±5% 이내
- [ ] 빈 행 없음 (최상단 행부터 채워져야 함)
- [ ] JSON 스키마 유효성 통과
```
