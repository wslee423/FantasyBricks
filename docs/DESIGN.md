## 타이포그래피

### 폰트
| 용도 | 폰트 | 크기 | 비고 |
|------|------|------|------|
| 게임 타이틀 | Noto Serif KR | 32sp | Bold, 골드 컬러 |
| 스테이지 제목 | Noto Serif KR | 20sp | SemiBold |
| 본문/설명 | Noto Sans KR | 14sp | Regular |
| HUD 수치 | Noto Sans KR | 16sp | Bold, Ivory |
| 버튼 텍스트 | Noto Sans KR | 15sp | SemiBold |

> Noto KR 계열은 한국어 전용, Google Fonts 무료 라이선스

### 텍스트 원칙
- 모든 인게임 텍스트는 **한국어**만 사용
- 아이템/스킬 이름은 판타지 한국어 표기 사용
  - ✅ "마법 관통석", "마법 분열구", "마법 화염포"
  - ❌ "Piercing Ball", "Multi Ball", "Laser"
- 버튼 텍스트는 2~4글자로 최대한 짧게

## 형태 & 스타일

### 기본 원칙
- **약간 둥근 모서리** — border-radius 8~12dp 기준
- 과도한 장식 배제, 심플한 실루엣 유지
- 판타지 느낌은 컬러와 이펙트로 표현, 형태는 심플하게

### 벽돌
- 기본 형태: 둥근 직사각형 (radius 8dp)
- 테두리: 1~2dp, 해당 HP 컬러보다 20% 밝은 색
- 아이템 드롭 벽돌: 중앙에 아이템 아이콘 (벽돌 크기의 40%)
- HP 감소 시: 균열 오버레이 이미지 단계별 적용 (1~4단계)

### 버튼
- Primary 버튼: Magic Gold 배경 + Deep Purple 텍스트
- Secondary 버튼: 투명 배경 + Gold 테두리 + Ivory 텍스트
- 비활성 버튼: Silver Mist 배경 + 50% 투명도

### 카드 (가챠 결과, 스킨 컬렉션)
- 배경: Royal Navy + 미세한 그라데이션
- 테두리: Arcane Violet, 1.5dp
- 전설 등급: 골드 글로우 이펙트 추가

## 이펙트 & 애니메이션

### 벽돌 파괴 이펙트
- 파편 4~6개 사방으로 퍼지며 페이드아웃
- 파편 색상: 해당 벽돌 HP 컬러 기준
- 지속 시간: 300ms

### 아이템 낙하
- 부드러운 위아래 흔들림 (bounce 애니메이션)
- 아이템 주변 작은 파티클 (색상: 아이템별 고유 컬러)
- 획득 시: 원형 파동 이펙트 + 사운드

### 아이템 활성화 이펙트
| 아이템 | 볼/패들 이펙트 | 궤적 이펙트 |
|--------|--------------|------------|
| 마법 관통석 | 파란 글로우 | 파란 잔상 |
| 마법 분열구 | 노란 글로우 | 노란 파티클 |
| 마법 화염포 | 패들 빨간 글로우 | 빨간 빔 라인 |

### UI 전환 애니메이션
- 화면 전환: 페이드 인/아웃 200ms
- 버튼 탭: scale 0.95로 수축 후 복귀 100ms
- 가챠 연출: 마법진 회전 → 카드 플립 → 결과 표시

## 화면 레이아웃

### 게임 플레이 화면 비율
```
┌─────────────────┐
│  HUD (10%)      │  ← 라이프, 아이템 상태, 스테이지
├─────────────────┤
│                 │
│  게임 영역 (75%) │  ← 벽돌 + 볼 + 패들
│                 │
├─────────────────┤
│  패들 조작 (15%) │  ← 터치 입력 영역
└─────────────────┘
```

### 안전 영역 (Safe Area)
- 상단: 상태바 + 16dp 여백
- 하단: 네비게이션바 + 16dp 여백
- 좌우: 12dp 여백
