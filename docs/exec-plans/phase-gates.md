# Phase Gates — 단계 전환 조건

각 Phase를 완료하고 다음 Phase로 넘어가기 위한 기준을 정의한다.
모든 gate 항목이 충족돼야 다음 Phase를 시작할 수 있다.

현재 Phase 업데이트는 `/sync-docs` 실행 시 자동으로 반영된다.

---

**Phase 4 — 진행 중**
