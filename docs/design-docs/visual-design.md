# Visual Design Guide — 중세 판타지 비주얼 가이드

## 테마 개요

Fantasy Bricks는 **중세 판타지 세계관**을 기반으로 한다.
단순한 RPG 껍데기가 아니라 마법·석재·어둠 분위기가 모든 UI에 녹아있어야 한다.
참고 키워드: 마법사의 탑, 흑요석, 수정, 룬 문자, 양피지 칙령

| HP | 이름 | 코드 | 의미 |
|----|------|------|------|

## 아이템 드롭 심볼

한글 텍스트를 사용하지 않는다. 색상 + 룬 심볼로 식별한다.

| 아이템 | 심볼 | 색상 | 의미 |
|--------|------|------|------|
| 마법 관통석 | `◈` | Ice Vein 파랑 | 꿰뚫는 날카로운 힘 |
| 마법 분열구 | `✦` | Sun Fragment 황금 | 폭발적 분열 에너지 |
| 마법 화염포 | `⚡` | Dragon Breath 빨강 | 화염과 번개의 파괴력 |

아이템 시각: 원형 gem — 외곽 링 + 내부 하이라이트 광택으로 보석 느낌 강조.

## 타이포그래피 원칙

- **헤더/타이틀**: `letterSpacing: 2~6`, `fontWeight: bold`
- **플레이버 텍스트**: `fontStyle: italic`, `letterSpacing: 1`
- **보조 정보**: `letterSpacing: 0.5`, 낮은 불투명도 색상
- **한글 사용 금지 위치**: 인게임 아이템 드롭, Flame 컴포넌트 내 텍스트

## UI 프레임 패턴

### 판타지 패널 (모달/오버레이)
```
┌─ ✦ ─────────────────── ✦ ─┐
│  (content)                  │
└─ ✦ ─────────────────── ✦ ─┘
```
- 외곽 테두리: `#B39DDB` 1px
- 배경: `#1A0A3D`
- 상하 장식선: 골드 `#FFD700` 50% 불투명도
- 코너 반경: `2` (날카롭지 않지만 원형도 아닌 중간)

### 인라인 HUD 바
- 배경: `#CC0D0620` (반투명 미드나이트)
- 테두리: 활성 아이템 색상 60% 불투명도
- 코너 반경: `4`

---

## 벽돌 3D 효과 규칙

- 상단/좌측 하이라이트: `white 18%` 두께 2.5px
- 하단/우측 그림자: `black 45%` 두께 2.5px
- 아이템 드롭 벽돌: Sacred Gold `50%` 1px 테두리 추가

## 적용 현황

| 구성요소 | 적용 여부 | 비고 |
|----------|----------|------|
| 벽돌 | ✅ | HP별 마법석 색상 + 3D 효과 |
| 아이템 드롭 | ✅ | 원형 gem + 룬 심볼 (한글 없음) |
| HUD 오버레이 | ✅ | 판타지 프레임 + 심볼 통일 |
| 승리 오버레이 | ✅ | 판타지 패널 + 장식선 |
| 패배 오버레이 | ✅ | 고딕 다크 스타일 |
| 로비 화면 | ✅ | 석성 타일 + 장식 헤더 |
| 가챠 화면 | ✅ | 페이드인 + 빛 번짐 이펙트 |
| 컬렉션 화면 | ✅ | 등급별 색상 프레임 |
| 볼 | 🔲 | 기본 단색; 스킨 시스템으로 커스터마이징 가능 |
| 패들 | 🔲 | 향후 스킨 시스템 확장 예정 |
| 배경 | 🔲 | 단색 Deep Void; 파티클 효과는 OD-017 |
