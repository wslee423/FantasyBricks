# PLANS.md — 개발 로드맵

## 개요
MVP를 최대한 빠르게 만들어 1~2일 내 테스트 가능한 상태로 만드는 것이
최우선 목표입니다. 총 10스테이지로 범위를 한정해 빠른 완성을 추구합니다.

---

## 전체 단계 요약

| 단계 | 목표 | 기간 |
|------|------|------|
| Phase 0 | 프로젝트 셋업 | 반나절 |
| Phase 1 | MVP — 플레이 가능한 핵심 루프 | 1~2일 |
| Phase 2 | 콘텐츠 확장 — 스테이지 10개 + 아이템 완성 | 1~2일 |
| Phase 3 | 가챠 + 컬렉션 + UI 완성 | 2~3일 |
| Phase 4 | 폴리싱 + 출시 준비 | 1~2주 |

- [ ] GitHub Actions 워크플로 설정 (빌드/테스트 자동화)
- [ ] fastlane 초기 설정 (Google Play 연동)

## Phase 1 — MVP (1~2일)

**목표**: 실제로 플레이 가능한 최소 버전
**테스트 기준**: APK 설치 후 1스테이지를 처음부터 끝까지 플레이 가능

### 포함 기능
- [ ] 게임 기본 루프
  - [ ] 볼 물리 (반사, 속도)
  - [ ] 패들 터치 조작
  - [ ] 볼 이탈 시 라이프 감소 + 패들 위 재생성 + 탭으로 발사
  - [ ] 벽돌 충돌 및 HP 감소
  - [ ] 벽돌 파괴 이펙트 (간단한 버전)
- [ ] 기본 아이템 1종 (마법 분열구 — 가장 직관적)
  - [ ] 아이템 드롭 벽돌
  - [ ] 아이템 낙하 및 획득
  - [ ] 마법 분열구 효과 발동
- [ ] 라이프 시스템 (3개, 볼 이탈 시 감소)
- [ ] 클리어 / 실패 판정
- [ ] 스테이지 1개 (JSON 하드코딩)
- [ ] 간단한 결과 화면 (클리어/실패 텍스트 + 재시작 버튼)

### 제외 기능 (Phase 1에서 의도적으로 뺌)
- 가챠, 컬렉션, 스킨
- 튜토리얼
- BGM / 효과음
- 로비 UI
- 로컬 저장소

### MVP 테스트 체크리스트
```
- [ ] APK 빌드 성공
- [ ] 패들 조작 자연스러운가
- [ ] 볼 반사각 자연스러운가
- [ ] 볼 이탈 후 패들 위 재생성 + 탭 발사 정상 작동
- [ ] 마법 분열구 효과 체감되는가
- [ ] 클리어/실패 판정 정상 작동
- [ ] 60fps 유지 (마법 분열구 활성화 시)
```

## Phase 2 — 콘텐츠 확장 (1~2일)

**목표**: 10스테이지 완성 + 아이템 3종 완성

- [ ] 아이템 시스템 완성
  - [ ] 마법 관통석 구현
  - [ ] 마법 화염포 구현 (정지 중에도 0.5초 간격 자동 발사)
  - [ ] 아이템 교체 로직 (중복 불가)
  - [ ] HUD 아이템 상태 표시
- [ ] 스테이지 10개 JSON 생성 (Stage Builder 에이전트)
  - [ ] 1~3 스테이지 (입문 — HP1~2, 드롭률 30%)
  - [ ] 4~7 스테이지 (중반 — HP2~4, 드롭률 20%)
  - [ ] 8~10 스테이지 (후반 — HP3~5, 드롭률 10%)
- [ ] 스테이지 선택 화면
- [ ] 로컬 저장소 연동 (진행도, 별점 저장)
- [ ] 튜토리얼 (Step 1~3)
- [ ] 효과음 + BGM 기본 버전
- [ ] 벽돌 HP별 시각 피드백 (균열 이미지)

> 30개 → 10개로 줄어 JSON 생성 + 밸런싱 시간이 대폭 단축됩니다.

## Phase 3 — 가챠 + UI 완성 (2~3일)

- [ ] 가챠 시스템
  - [ ] 마법 코인 획득/소모 로직
  - [ ] 뽑기 확률 테이블 (일반 70% / 희귀 25% / 전설 5%)
  - [ ] 가챠 연출 애니메이션 (마법진 소환)
  - [ ] 파편 시스템
- [ ] 컬렉션 화면
  - [ ] 보유 스킨 목록
  - [ ] 스킨 장착/해제
  - [ ] 파편 교환소
- [ ] 볼 스킨 적용 (인게임 반영)
- [ ] 이펙트 스킨 적용
- [ ] 로비 화면 완성
- [ ] 광고 연동 (AdMob)
  - [ ] 실패 시 광고 보고 라이프 1 회복
  - [ ] 코인 획득 보상형 광고 (일 5회 제한)

## Phase 4 — 폴리싱 + 출시 준비 (1~2주)

**목표**: Google Play 정식 출시

- [ ] 전체 QA 테스트
  - [ ] 스테이지 10개 클리어 가능 여부 검증
  - [ ] 볼 재시작 로직 엣지 케이스 테스트
  - [ ] 메모리 누수 확인
  - [ ] 다양한 Android 해상도 테스트
- [ ] 앱 아이콘 + 스플래시 스크린
- [ ] Google Play 스토어 등록
  - [ ] 앱 설명 (한국어)
  - [ ] 스크린샷 5장
  - [ ] 피처드 이미지
- [ ] 개인정보처리방침 페이지 (광고 SDK 필수)
- [ ] 내부 테스트 → 비공개 테스트 → 정식 출시
