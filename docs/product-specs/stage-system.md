# Stage System

## 개요
총 10개의 스테이지로 구성된 스토리 진행형 게임입니다.
모든 벽돌을 파괴하면 클리어, 라이프가 0이 되면 실패입니다.

---

## 클리어 / 실패 조건

| 조건 | 내용 |
|------|------|
| 클리어 | 스테이지 내 모든 벽돌 파괴 |
| 실패 | 라이프 0 도달 시 게임오버 (클리어 불가) |
| 라이프 감소 | 볼이 하단 경계 이탈 시 라이프 -1 |
| 기본 라이프 | 3개 (하트 아이콘으로 표시) |

---

## 별점 기준

| 클리어 시 남은 라이프 | 별점 |
|-------------------|------|
| 3개 (한 번도 이탈 없음) | ⭐⭐⭐ |
| 2개 | ⭐⭐ |
| 1개 | ⭐ |
| 0개 | 실패 (별점 없음, 클리어 불가) |

> 라이프가 0이 되면 클리어가 불가능하므로 0성은 존재하지 않습니다.
> 최저 클리어 별점은 1성 (라이프 1개 남김)입니다.

---

## 스테이지 구성 및 난이도 커브

| 구간 | 스테이지 | 테마 | 특징 |
|------|---------|------|------|
| 입문 | 1~3 | 숲 / 초원 | HP1~2 벽돌 위주, 아이템 드롭 벽돌 多 |
| 중반 | 4~7 | 던전 / 마법사 탑 | HP2~4 혼합, 배치 패턴 다양화 |
| 후반 | 8~10 | 암흑 성채 / 마왕의 방 | HP3~5 위주, 복잡한 패턴, 드롭률 低 |

---

## 스테이지 구성 요소

### 그리드 구조
- 화면 상단 70%를 벽돌 배치 영역으로 사용
- 기본 그리드: 7열 × 10행 (최대 70개 벽돌)
- 모든 셀을 채우지 않고 패턴/모양으로 배치

### 벽돌 배치 패턴 예시
- **입문 (1~3)**: 단순 직사각형 배열
- **중반 (4~7)**: 체커보드, 다이아몬드형
- **후반 (8~10)**: 성, 마법진 모양 등 판타지 테마 실루엣

### 아이템 드롭 벽돌 출현 비율
| 구간 | 스테이지 | 드롭 비율 |
|------|---------|----------|
| 입문 | 1~3 | 30% |
| 중반 | 4~7 | 20% |
| 후반 | 8~10 | 10% |

> 후반으로 갈수록 아이템 의존이 어려워져 실력 기반 플레이 유도

---

## 스테이지 클리어 보상

| 보상 | 내용 |
|------|------|
| 별점 | 1~3성 평가 (라이프 소모량 기준) |
| 마법 코인 | 클리어 시 10코인 지급 |
| 첫 클리어 보너스 | 최초 클리어 시 +5코인 추가 |

---

## 게임오버 & 재시작

- 라이프 0 → 게임오버 화면 표시
- 선택지: **재시작** / **광고 보고 라이프 1 회복** / **로비로 이동**
- 클리어한 스테이지는 언제든 재도전 가능 (별점 갱신)
