# New User Onboarding

## 개요
첫 실행 시 인터랙티브 튜토리얼을 통해 핵심 조작법과
아이템 시스템을 자연스럽게 익히도록 유도합니다.
스킵 버튼을 제공하여 경험자의 진입 장벽을 최소화합니다.

---

## 튜토리얼 진입 조건

- 최초 앱 실행 시 자동 시작
- 우측 상단 **[SKIP]** 버튼으로 언제든 건너뛰기 가능
- 설정 메뉴에서 튜토리얼 재시작 불가 (간소화 목적)

---

## 튜토리얼 구성 — 총 3단계

### Step 1. 패들 조작 익히기
- **화면 구성**: 벽돌 4개 (HP1, 일반 벽돌), 볼 1개
- **힌트 표시**: 손가락 아이콘 + 텍스트
  > "손가락을 좌우로 움직여 패들을 조종하세요"
- **진행 조건**: 벽돌 4개 모두 파괴 시 다음 단계로
- **실패 없음**: 볼이 떨어져도 자동 리셋 (라이프 차감 없음)

---

### Step 2. 아이템 획득 & 효과 체험
- **화면 구성**: 벽돌 6개 (HP1~2 혼합), 아이템 드롭 벽돌 2개
  - 드롭 아이템: 마법 분열구 1개, 마법 관통석 1개 고정 배치
- **힌트 흐름**:
  1. 아이템 드롭 벽돌에 화살표 + 강조 효과
     > "이 벽돌을 깨면 아이템이 떨어져요!"
  2. 아이템 낙하 시 패들 방향 유도 화살표 표시
     > "패들로 아이템을 받으세요!"
  3. 아이템 획득 직후 효과 설명 팝업
     > "마법 분열구 획득! 볼이 3개로 분열됩니다"
- **진행 조건**: 아이템 1개 이상 획득 + 모든 벽돌 파괴 시 완료

---

### Step 3. 튜토리얼 완료 & 보상
- **완료 연출**: 마법진 이펙트 + "모험을 시작할 준비가 됐어요!" 텍스트
- **보상 지급**: 마법 코인 30개 (튜토리얼 완료 시에만 지급)
- **이후 흐름**: 가챠 1회 체험 유도 팝업 → 가챠 화면 → 로비 → 1스테이지 진입

---

## 스킵 시 처리

- 스킵 선택 시 확인 팝업 없이 즉시 건너뜀
- **스킵 시 보상 미지급** — 마법 코인 30개는 튜토리얼 완료 시에만 지급
  > 스킵은 게임을 이미 아는 경험자를 위한 기능이므로
  > 보상 없이 바로 1스테이지로 진입합니다.
- 가챠 체험 안내는 튜토리얼 완료자에게만 표시

---

## 튜토리얼 UX 원칙

| 원칙 | 내용 |
|------|------|
| 설명 최소화 | 텍스트보다 화살표/강조 효과로 유도 |
| 실패 부담 없음 | 볼 이탈 시 라이프 차감 없이 자동 리셋 |
| 즉각적 피드백 | 아이템 획득 즉시 효과 팝업으로 학습 강화 |
| 빠른 진입 | 3단계 전체 소요 시간 목표: 90초 이내 |
| 가챠 연계 | 완료 보상으로 자연스럽게 가챠 첫 경험 유도 |
