# Item System

## 개요
아이템은 아이템 드롭 벽돌 파괴 시 낙하하며,
패들로 획득하면 즉시 효과가 발동됩니다.
한 번에 1개만 활성화되며, 새 아이템 획득 시 기존 효과는 즉시 교체됩니다.

---

## 공통 규칙

| 규칙 | 내용 |
|------|------|
| 중복 | 불가 — 새 아이템이 기존 아이템을 즉시 교체 |
| 지속 방식 | 볼 충돌 횟수 기준 (기본값: 20회 타격) |
| 만료 시 | 효과 종료, 볼/패들 외형 원상복구 |
| 획득 방법 | 낙하 중인 아이템을 패들로 터치/접촉 |
| 소멸 조건 | 패들에 닿지 못하고 하단 경계 도달 시 소멸 |

---

## 볼 이탈 및 재시작 규칙

| 상황 | 처리 |
|------|------|
| 일반 볼 하단 이탈 | 라이프 -1 → 아이템 효과 초기화 → 볼 패들 위에 재생성 |
| 재생성 후 발사 | 플레이어가 화면 탭 시 발사 (자동 발사 없음) |
| 재생성 후 무적 시간 | 없음 |
| 멀티볼 중 볼 1개 이탈 | 라이프 차감 없음, 나머지 볼 계속 진행 |
| 멀티볼 중 모든 볼 이탈 | 라이프 -1 → 아이템 효과 초기화 → 볼 1개 패들 위 재생성 |
| 라이프 0 도달 | 게임오버 |

## 아이템 상세 정의

### 🔵 마법 관통석 (Arcane Pierce)
- **효과**: 볼이 벽돌에 충돌해도 튕기지 않고 관통하며 연속 타격
- **지속**: 20회 타격
- **시각 피드백**:
  - 볼 색상 → 파란색 + 빛나는 이펙트
  - UI 상단: `마법 관통석 | 남은 타격 N회`

### 🟡 마법 분열구 (Arcane Split)
- **효과**: 현재 볼이 3개로 분열, 모든 볼이 동시에 활동
- **지속**: 분열된 볼 기준 합산 20회 타격
- **추가 규칙**:
  - 분열된 볼 중 일부가 하단 이탈해도 나머지 볼 유지 (라이프 차감 없음)
  - 모든 볼이 하단 이탈하면 라이프 -1, 볼 1개로 재시작
  - 재시작 시 아이템 효과 초기화
- **시각 피드백**:
  - 볼 3개 모두 노란색으로 변경
  - UI 상단: `마법 분열구 | 남은 타격 N회`

### 🔴 마법 화염포 (Arcane Cannon)
- **효과**: 패들 중앙에서 위쪽으로 레이저 발사, 일직선상 모든 벽돌 즉시 1 데미지
- **지속**: 20회 타격 (레이저 발사 1회 = 타격 1회로 계산)
- **발동 방식**:
  - 패들 이동 중 자동 연속 발사 (0.5초 간격)
  - 패들 정지 시에도 0.5초 간격으로 계속 발사 (이동 여부 무관)
- **시각 피드백**:
  - 패들 색상 → 빨간색 + 레이저 빔 이펙트
  - UI 상단: `마법 화염포 | 남은 발사 N회`

## UI 상단 아이템 HUD

```
[ 🔵 마법 관통석 | ██████░░░░ | 남은 타격: 12회 ]
```

- 아이템 아이콘 + 이름 + 프로그레스 바 + 남은 횟수
- 아이템 없을 시 해당 영역 비워둠

---

## 아이템 교체 시나리오

1. 마법 관통석 활성화 중 (남은 타격 8회)
2. 플레이어가 마법 분열구 아이템 획득
3. 마법 관통석 즉시 종료 → 볼 외형 원상복구
4. 마법 분열구 즉시 발동 → 볼 3개 분열, HUD 업데이트
