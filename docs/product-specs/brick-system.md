# Brick System

## 개요
벽돌은 게임의 핵심 상호작용 단위입니다.
HP 수치와 아이템 드롭 여부로 벽돌의 특성을 정의합니다.

---

## 벽돌 기본 속성

| 속성 | 설명 |
|------|------|
| `hp` | 1~5 사이의 정수. 볼 충돌 1회당 1 감소 |
| `item_drop` | null 또는 아이템 ID. 파괴 시 드롭 여부 결정 |
| `color_tier` | HP에 따라 시각적으로 구분 (HP1=밝은색 → HP5=어두운색) |
| `position` | 그리드 좌표 (col, row) |

---

## 벽돌 종류

### 1. 일반 벽돌 (Normal Brick)
- HP: 1~5
- 아이템 드롭: 없음
- 역할: 스테이지 배경 구성, 난이도 조절

### 2. 아이템 드롭 벽돌 (Item Brick)
- HP: 1~5
- 아이템 드롭: 있음 (파괴 시 스킬 아이템 1개 낙하)
- 시각 표시: 벽돌 내부에 아이템 아이콘 표시
- 드롭되는 아이템 종류:
  - 🔵 **마법 관통석** — 볼이 벽돌을 관통하며 연속 타격
  - 🟡 **마법 분열구** — 볼 3개로 분열
  - 🔴 **마법 화염포** — 패들에서 위쪽으로 레이저 발사, 일직선 벽돌 즉시 1 데미지

---

## HP별 타격 시각 피드백

| HP | 색상 | 균열 표현 |
|----|------|----------|
| 5 | 다크 인디고 (#2D1B69) | 없음 |
| 4 | 딥 퍼플 (#4A0E8F) | 균열 1단계 |
| 3 | 로열 퍼플 (#7B2FBE) | 균열 2단계 |
| 2 | 미디엄 퍼플 (#9B59B6) | 균열 3단계 |
| 1 | 연한 라벤더 (#C8A8E9) | 금방 깨질 듯한 균열 |

---

## 드롭 로직

1. 벽돌 HP가 0이 되는 순간 파괴 이펙트 재생
2. `item_drop`이 null이 아니면 해당 아이템을 벽돌 위치에서 아래로 낙하 시작
3. 패들 영역까지 낙하 중 플레이어가 패들로 받으면 즉시 효과 발동
4. 패들에 닿지 못하면 아이템 소멸
