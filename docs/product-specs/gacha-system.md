# Gacha System

## 개요
가챠는 순수 코스메틱 + 이펙트 강화 중심입니다.
게임 실력에 영향을 주지 않으며, 플레이어의 개성 표현 수단입니다.
재화는 광고 시청 및 스테이지 클리어 보상으로만 획득 가능한 완전 무료 모델입니다.

---

## 재화 시스템

### 재화명: 마법 코인 (Arcane Coin)

| 획득 방법 | 획득량 |
|----------|--------|
| 스테이지 첫 클리어 | 10코인 |
| 스테이지 재클리어 (별점 향상 시) | 5코인 |
| 3성 클리어 보너스 | +3코인 추가 |
| 광고 시청 | 5코인 / 1회 (일 5회 제한, 확정) |

---

## 뽑기 방식

- **1회 뽑기**만 지원
- 뽑기 비용: 30코인 / 1회
- 중복 획득 시: 자동으로 파편으로 전환
- 파편 30개 → 원하는 아이템 1개 교환 가능

---

## 가챠 대상

### 1. 볼 스킨 (Ball Skin)
게임 플레이 중 볼의 외형을 변경합니다.
아이템 활성화 시에도 스킨 기반 색상 위에 이펙트 오버레이 적용.

| 등급 | 확률 | 예시 |
|------|------|------|
| 일반 | 70% | 단색 볼 (빨강, 파랑, 초록 등) |
| 희귀 | 25% | 불꽃 볼, 얼음 볼, 번개 볼 |
| 전설 | 5% | 마왕의 눈, 별똥별, 마법진 볼 |

### 2. 아이템 이펙트 강화 (Effect Skin)
마법 관통석 / 마법 분열구 / 마법 화염포 활성화 시 시각 이펙트를 업그레이드합니다.
**성능 변화 없음** — 순수 시각 효과만 변경.

| 아이템 | 기본 이펙트 | 강화 이펙트 예시 |
|--------|-----------|----------------|
| 마법 관통석 | 파란 빛 | 빙결 균열 / 번개 궤적 / 마법진 잔상 |
| 마법 분열구 | 노란 분열 | 불꽃 폭발 / 성운 분열 / 황금 궤적 |
| 마법 화염포 | 빨간 빔 | 용의 숨결 / 마법 봉인 빔 / 어둠의 광선 |

## 가챠 UI 흐름

```
[로비] → [가챠 버튼] → [보유 코인 확인]
→ [뽑기 연출 (마법진 소환)]
→ [결과 화면: 획득 아이템 표시]
→ [중복 시 파편 전환 안내]
→ [컬렉션 화면으로 이동 or 로비로 복귀]
```

---

## 컬렉션 화면

- 획득한 볼 스킨 / 이펙트 스킨 목록 표시
- 미획득 항목은 실루엣 + 물음표로 표시
- 현재 장착 중인 스킨 하이라이트 표시
- 파편 현황 및 교환소 접근 가능

---

## 수익화 원칙

- **인앱결제 없음** — 광고 기반 완전 무료
- 광고 강제 시청 없음 — 모두 자발적 선택
- 가챠 결과가 게임 승패에 영향 없음 (Pay-to-Win 배제)
