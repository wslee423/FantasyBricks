# Product Specs — Index

## 게임 개요
- **장르**: 벽돌깨기 + 아이템 수집형 캐주얼 게임
- **플랫폼**: Android 우선 (iOS 추후 확장)
- **테마**: 판타지/RPG
- **핵심 차별점**:
  - 벽돌 파괴 시 특수 스킬 아이템 즉시 드롭
  - 가챠 시스템 (볼 스킨, 이펙트 커스터마이징)
